## How to Use

### Option 1: Add All Subjects at Once
1. Click **"Add All 19 Preset Subjects"** button
2. All 19 subjects will be added to the table
3. Fill in the question details for each subject directly in the table

### Option 2: Add Subjects One by One
1. Type subject name in the input field
2. Autocomplete will show matching preset subjects
3. Select from dropdown or type custom name
4. Fill in question details
5. Click "Add Subject"

### Option 3: Edit in Table
- After adding subjects, you can edit them directly in the table
- Click on any number field to edit
- Changes save automatically
